# remato
REMATO - REssource MAnagement TOol

Remato is a management tool for Austrian ambulance stations.
You will be able to manage:
- Trainees
- Vehicles
- Medical Devices
- Daily Jobs

### Built With

* [.NET5](https://github.com/dotnet)
* [Talaryon Toolbox](https://github.com/talaryonlabs/toolbox)
